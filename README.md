# Img2sh

Img2sh is a tool to show images directly on terminal

### Installing

### Demo

```
img2sh demo.jpeg
```

```
usage: img2sh [-h] [-w WIDTH] Image

Show images directly on terminal.

positional arguments:
  Image

optional arguments:
  -h, --help            show this help message and exit
  -w WIDTH, --width WIDTH
```

### Development

#### Setup development environment
```
pipenv install --dev
```

```
pipenv dev
```
